# Global-Sales-Performance-Dashboard

## Overview
This project is an interactive Excel dashboard built to analyze over **113,000 sales records** and uncover insights on revenue, profitability, and regional performance.
The project demonstrates practical data analysis skills using Excel for cleaning, transforming, and analyzing real-world business data.

## Objectives
- Analyze sales performance across regions and markets  
- Identify high-profit regions and product categories  
- Track revenue trends over time  

## Tools Used
- Microsoft Excel  
- Power Query (data cleaning and transformation)  
- Pivot Tables  
- Charts and Slicers  

## Key Insights
- Top-performing regions by revenue  
- Most profitable product categories  
- Sales trends and performance patterns
